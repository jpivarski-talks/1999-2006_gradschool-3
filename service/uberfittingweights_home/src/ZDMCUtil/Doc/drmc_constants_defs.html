Overview of DR MC constants

See DRMCConstants.html for an overview, links to the Fortran access functions, constants BDL files, ASCII FileSourceFormat libraries, and the ASCII constants files.

DRMC Anode Constants Definitions

  • DRMCCoseff - 2D fit of Cos(theta) vs drift - 47 entries
    • Min - 1st polynomail coeff. for drift projection
    • Max - 2nd polynomail coeff. for drift projection
    • Cheb1 - 1st polynomail coeff. for cos(theta) projection
    • Cheb2 - 2nd polynomail coeff. for cos(theta) projection

  • DRMCHitJump - Probability a hit will jump to the next cell - 47 entries. Used in CDHITS (which called movhit in Cleo2).
    • Probn - negative probability
    • Probp - positive probability
    • Slpn - Slope -
    • Slpp - Slope +
    • Dstn - linear parameter
    • Dstp - linear parameter

  • DRMCIons - Ion statistics important at center of cell - 47 entries. Store number of ions required to trigger cell. DOCA not equal to the true distance for the ion pair that fires the cell. function of drift distance & entrance angle.
    • N

  • DRMCLyreff - drift view of drift vs. cos(theta) - 47 entries
    2 legs on side + 2nd order function for the center of the cell with a "tooth" in the center.
    • Slpn
    • Slpp
    • Dstn
    • Dstp
    • Norm
    • Chb1
    • Chb2
    • Slpm
    • Xr

  • DRMCPhlayr - layer Pulse height conversion factor - 940 entries
    20 bins in drift distance per layer x 47 layers
    • Lyrcor

  • DRMCPhwire - Wire to wire pulse height corrections - 9796 entries
    # of sense wires
    • Wircor

  • DRMCResolutions - cell resolutions - 2000 entries In bins of drift distance, 20 bins (0-19) per layer. Offset = 100 * bin + layer#. Current ascii file contains constants for all DR2 layers - easy enough just to leave them in.
    • Sigma1 - Width of Gaussian in microns
    • Sigma2 - not used
    • Offst1 - shifted drift distance
    • Offst2 - not used
    • Frac - linear smear across cell, used to tune Duet eff.

  • DRMCSpefud - particle species dependent eff. fudge - 47 entries
    • Elefud - implemented for electrons, but machinary in place for others
    • Muofud
    • Piofud
    • Kaofud
    • Profud

  • DRMCWireff - overall efficiency for each wire - 9796 entries
    • Effi

DRMC Cathode Constants Definitions

  • DRMCCathname- Partial filename of cathode histogram - 1 entry
    • Name - contains part of the file name to load cathode histograms, CLEO-II stored these in /ccon/ccon/4spcath.rzn
    • This is stupid - put entire filename in constant
    • Better yet, don't use histograms to store efficiency

  • DRMCEff - Cathode efficiency vs phi - 1 entries
    • A - sine wave parameters vs phi to describe radiation damage
    • B - sine wave parameters vs phi to describe radiation damage
    • C - sine wave parameters vs phi to describe radiation damage

  • DRMCSmear - cathode resolution in cm - 1 entries
    • DZ

  • DRMCTmlayr - Cathode times - 1 entry
    Obsolete: no cathode timing info in cleo3
    • Shift
